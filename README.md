# Rodentia Platform

**Спеціалізована CRM-система для викладачів-фрілансерів, яка перетворює хаос у месенджерах та блокнотах на впорядковану екосистему.**

Rodentia допомагає викладачам автоматизувати рутину: від фінансового обліку до контролю знань учнів. Система об'єднує розклад, оплати та навчальні матеріали в одному місці, роблячи процес навчання **структурованим, прозорим та професійним**.

---

## Ключові особливості системи

- **Фінансовий монітор «NutControl»** — швидка відмітка оплати занять та автоматичний розрахунок заборгованостей.
- **Інтелектуальний розклад** — зручне керування уроками, перевірка накладок у часі та гнучка система статусів (заплановано/проведено/скасовано).
- **Модуль знань** — журнал для моніторингу прогресу, зручна передача домашніх завдань та смарт-бібліотека матеріалів.
- **Безпека та доступ** — чітке розділення ролей між викладачем та учнем, захищений вхід та шифрування даних.
